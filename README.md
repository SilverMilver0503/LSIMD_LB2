### Вербальний опис моделі:
Модель симулює спрощений парниковий ефект Землі. Сонячні промені падають на поверхню, частина відбивається (залежно від альбедо та хмарності), решта поглинається й перетворюється в тепло. Земля нагрівається, випромінює інфрачервоні (IR) хвилі, які можуть або втекти в космос, або відбитися від молекул CO_2 і повернутись до поверхні, підіймаючи середню температуру. Таким чином, баланс між сонячним випромінюванням, альбедо, кількістю хмар і концентрацією CO_2 визначає, наскільки “тепло” утримується в системі.

### Внесені зміни в код:

- Було додано нові вхідні віджети (на зображенні нижче)
![0](1.png)
- Логіку зміни дня і ночі 
``` 
to update-sun-angle
  ;; якщо day-length некоректний, просто тримаємо центр
  if day-length <= 0 [
    set sun-angle sun-angle-center
    stop
  ]

  ;; синусоїдальний хід сонця протягом "доби"
  let phase (2 * pi * ticks) / day-length
  let a sun-angle-center + sun-angle-amplitude * sin phase

  ;; тримаємо промінь спрямованим донизу (щоб не летів у небо)
  set sun-angle clamp a 100 260
end
- Промені розтягнуті рівномірно по екрану - було додано ще під час попередньої лабораторної роботи
```

## Обчислювальний експеримент

Як можна побачити ще на першому рисунку, промені падають під одним кутом. З плином часу кут змінюється

![0](2.png)

На графіку можна побачити підйоми і падіння температури, пов'язані зі зміною дня і ночі.
Але цікаво, що температури припадає на зеніт сонця - коли промені перпендикулярні землі. Це пов'язано з акумулюванням тепла, через що створюється певна "інерція" температури. Детальніше цей ефект я готовий пояснити під час захисту лабораторної роботи.
